# Security Policy

## Supported Versions

PicForge is pre-1.0. Security fixes land on the active development branch.

## Reporting A Vulnerability

Please do not open a public issue for a security report. Contact the maintainer privately through the repository contact channel.

Include:

- Affected version or commit.
- Browser and operating system.
- Steps to reproduce.
- Impact and any suggested mitigation.

## Privacy Model

PicForge processes images locally in the browser. The app does not require accounts, server-side image processing, or analytics to compress files. Contributors should preserve this model.
